Password Generator

node-pwgen

Easy-to-remember secure password generator, which generates passwords such as SUPER.Obedient.FISH856 and ..INWARDLY.Sweet.Ostrich595.. in the fashion of {adverb|adjective} adjective animal{number}, with special character padding. Generation can be customized such as adding padding, changing the number of numbers, and declaring a maximum length, in which case you might end up with a password like -ICY-Hard-CAT58-

See source code and more at github.com/vartan/node-pwgen
